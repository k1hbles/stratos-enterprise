# Nightshift UI Template

Saved snapshot of the Nightshift v1 interface before the Hyprnova redesign.

## What's Included
- Complete Next.js frontend (App Router)
- Dark theme design system with glassmorphism
- Sidebar navigation with collapsible layout
- Chat interface (Kimi-inspired agent step cards)
- Workspace split-pane view
- Deliverable cards with download
- Tailwind CSS configuration
- TypeScript types

## Quick Start
```bash
cp -r ~/nightshift-ui-template my-new-project
cd my-new-project
npm install
npm run dev
```

## Design References
See DESIGN_SYSTEM.md for full color palette, component patterns, and spacing guidelines.

## Git Recovery
If you need the full project (including backend):
```bash
cd ~/your-nightshift-repo
git checkout nightshift-v1-ui
```
